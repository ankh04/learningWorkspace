### 快速打印的几种方法
```java
// sout  
System.out.println();  
// soutp  
System.out.println("args = " + Arrays.deepToString(args));  
// soutm  
System.out.println("Hello.main");  
  
int a = 10;  
// soutv  
System.out.println("a = " + a);  
  
// xxx.sout  
System.out.println(a);  
// xxx.soutv  
System.out.println("a = " + a);
```
